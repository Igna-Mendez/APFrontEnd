Configuración del Formulario

Para que este formulario le funcione en forma correcta debera completar algunos campos:

Nombre de Campo Descripción
_recipients Necesario. Donde se enviara el correo con la suscripción. Multiple recipients pueden ser utilizados, y deberan ser separados por comas. (','). 

  <input name="_recipients" type="hidden" value="webmaster@abc.net">
  OR
  <input name="_recipients" type="hidden"
   value="sales@xyz.com, orders@xyz.com">
_replyTo Cuenta de correo que sera utilizada en la cabecera del correo para la respuesta. 

  <input name="_replyTo" type="hidden" value="techsupt@xyz.com">
_replyToField Igual _replyTo salvo que ud especifica el nombre de otro campo del formulario. Generalmente se utiliza para el campo donde se hace referencia al correo del que se suscribe. 

  <input name="_replyToField" type="hidden" value="Email Address">
  ...
  Your email address: <input name="Email Address" type="text" size=20>
_subject Especifica el asunto del correo. 

  <input name="_subject" type="hidden" value="Site Feedback">
_requiredFields Una coma delimita la cantidad de campos que debería chequear el formmail. Cualquier valor que falte causara un alerta de error evitando la suscripción. 

  <input name="_requiredFields" type="hidden"
  value="Nombre, Direccion, Ciudad, Estado, Codigo Postal, Correo">
_fieldOrder Una coma delimita la lista de campos. Cuando se genere el correo, los campos y los valores se mostraran en el orden que ud los coloque. 

  <input name="_fieldOrder" type="hidden"
  value="Nombre, Correo, Teléfono, Dirección, Ciudad, Estado, Código Postal">

_redirect Normalmente cuando se envia el correo, se muestra una pagina comentando esto, pero si di desea puede colocar una pagina de agradecimiento que ud desee. 

  <input name="_redirect" type="hidden" value="termsofservice.html">
