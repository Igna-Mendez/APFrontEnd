Datos del Buscador

En el directorio CGI-BIN tiene una carpeta (buscador) con un buscador el cual debera colocar todos sus archivos (4) en el lugar mas alto desde donde desea comenzar a buscar. Luego desde la pagina que desea mostrar el buscador debera hacer referencia la archivo buscador.asp El buscador buecara las palabras relacionadas en todos los archivos del sitio, con lo cual no necesitara realizar indexaciones de bases.


Datos del Administrador de Banners

En el directorio CGI-BIN tiene una carpeta (Banner) con un administrador de banners. Para su correcto funcionamiento debera poner en la pagina que desee el buscador el siguiente codigo:

<HTML>
<HEAD>
<TITLE>Random Banner from DB Example</TITLE>
<!--#INCLUDE FILE="cgi-bin/banner/banners.shtml"-->
</HEAD>
<BODY>
<%
Rnd_Banner()
%>
</BODY>
</HTML>


En http://www.sudominio/cgi-bin/banner/admin/ encontrara un panel de control del administrador de banner en el cual ud podra subir las imagenes necesarias para que su funcionamiento sa exitoso. Las imagenes se instalaran en una base de datos Access.


Contador Gráfico

Esta aplicacion es sencilla, solo debera colocar el siguiente codigo en la pagina donde desea que le aparezca el contador. El codigo es el siguiente:

<!--#INCLUDE FILE="./cgi-bin/counter/contador.asp"-->

Esto le permitira ver su contador personal. Para resetear las visitas o setearlas debera pornerlas en el archivo "contador.txt" que se encuentra en /cgi-bin/counter/.

Lista de Correo (Mailling List)

El mailing List fue instalado y se encuetra dentro de la carpeta cgi-bin con el nombre mlm Para acceder por web a la misma debera hacerlo por http://www.sudominio/cgi-bin/mlm/ Allí verá el archivo (subscri.asp) de suscripcion donde la persona que desee suscribirse debera realizarlo a la lista que le aparecera arriba cuando ud, ya haya creado las mismas. En http://www.sudominio/cgi-bin/mlm/admin/ colocando su usuario y pass le aparecera la manera en la cual ud podra realizar las respectivas actualizaciones de la lista de correo. Cada passo esta comentado para una mejora administracion.
